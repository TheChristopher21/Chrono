# Chrono-Logisch E-Mail-Vorlage

Die HTML-Datei enthält eine geschäftliche Standardnachricht mit vollständiger Signatur und Firmenlogo. Alle eckig geklammerten Texte sind Platzhalter und werden vor dem Versand ersetzt.

## Empfohlener Betreff

`[Thema] – Chrono-Logisch`

Beispiele:

- `Ihre Anfrage zur Zeiterfassung – Chrono-Logisch`
- `Terminbestätigung für den 21. Juli 2026 – Chrono-Logisch`
- `Ihr individuelles Angebot – Chrono-Logisch`

## Verwendung in Gmail

1. `chrono-geschaeftlich.html` in einem Browser öffnen.
2. Den sichtbaren E-Mail-Inhalt markieren und kopieren.
3. In eine neue Gmail-Nachricht einfügen.
4. Anrede, Platzhalter und Betreff anpassen.

Das Logo wird über `https://chrono-logisch.ch/img/komplettesLogo.png` geladen. Dadurch bleibt es auch beim Empfänger sichtbar. Die Textdatei dient als schlichte Alternative für E-Mail-Programme ohne HTML-Formatierung.
